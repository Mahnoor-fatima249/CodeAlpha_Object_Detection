# Object Detection and Tracking - Task 4 👁️

## Overview
This project implements real-time object detection and tracking using OpenCV and Mediapipe. It can detect and track human hands/objects via a live webcam feed.

## Features
- **Real-time Detection**: Uses webcam for instant feedback.
- **Hand Tracking**: Precisely tracks hand landmarks and connections.
- **OpenCV Integration**: High-performance image processing.

## How to Run
1. Install requirements: `pip install opencv-python mediapipe`
2. Run command: `python app.py`
